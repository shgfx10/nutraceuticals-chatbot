## Nutraceutical AI Analyst Chatbot

An evidence-based chatbot designed to provide fast, reliable insights on dietary supplements, functional foods, and health ingredients. Powered by the Gemini API and Search Grounding.

---

##  Live Demo

You can try the live version of the chatbot here:
[**Launch Chatbot**](https://shgfx10.github.io/nutraceuticals-chatbot/)

---

##  Features

* **Evidence-Based Answers:** Uses Google Search grounding to ensure information is current and accurate.
* **Concise Summaries:** Provides facts, benefits, dosage considerations, and potential side effects.
* **User-Friendly Interface:** Built with plain HTML, JavaScript, and styled using Tailwind CSS.

---

##  Technology Stack

* **Core Logic:** Google Gemini API (`gemini-2.5-flash-preview`)
* **Front-end:** HTML, Vanilla JavaScript
* **Styling:** Tailwind CSS
